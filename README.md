# 🎨 Skill Design — O arsenal de skills do Claude Code pra melhorar páginas

Curso-catálogo, no formato **INEMA.CLUB**, que analisa os **16 skills** do Claude Code para **criar e melhorar páginas**:
o que cada skill faz, quando dispara, como melhora o resultado, como funciona por dentro, prompts prontos e com quais skills ele combina.

## 🌐 Acesse o curso

**https://inematds.github.io/skill-design/**

## 📚 As 4 trilhas (16 skills)

| Trilha | Tema | Skills analisados |
|--------|------|-------------------|
| 🏗️ **T1 — Construir interfaces** | Gerar e redesenhar UI | `frontend-design` · `impeccable` · `web-artifacts-builder` · `design-designer` |
| 🎨 **T2 — Identidade & temas** | Look & feel coerente | `theme-factory` · `brand-guidelines` · `canvas-design` · `algorithmic-art` |
| 🎞️ **T3 — Mídia & vídeo** | Movimento e vídeo na página | `animation-designer` · `3d-animation-creator` · `remotion` · `seedance` + `HyperFrames`/`FFmpeg` |
| 🔎 **T4 — Diagramas, navegador & pesquisa** | Apoio à página | `beautiful-mermaid` · `excalidraw` · `agent-browser` (Playwright) · `website-intelligence` |

Cada módulo é a **análise de um skill** em 6 ângulos: o que faz · quando dispara · como melhora suas páginas ·
como funciona por dentro · exemplo + prompt pronto · combina com & limites.
